# PrimeMart Retail Sales Analysis (Excel)

End-to-end Excel analysis of a retail dataset for a fictional Nigerian retailer, PrimeMart. Covers data cleaning, formulas, pivot tables, charts, and an executive dashboard, built over 7 days.

## Dataset

Six related sheets, about 5,000 transactions total: Orders, Customers, Products, SalesReps, Returns, Shipping.
